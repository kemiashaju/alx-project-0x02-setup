# alx-project-0x02-setup
Next.js Project Setup and Basics

This repository, alx-project-0x02-setup, is dedicated to setting up the foundational structure for a Next.js application as part of the ALX Frontend Learning Path.
It contains the initial project configuration, essential folders, and basic files needed to begin working with Next.js.
The goal of this repository is to help you understand how to properly initialize, configure, and structure a Next.js project from scratch.
